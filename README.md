# 国家免疫规划

本项目是使用Nuxt3开发的国家免疫规划科普网页，旨在完成《文明之痕：流行病与公共卫生》的期末考核

## 如何启动本项目

1. 安装nodejs

具体方法可以参考[官方介绍](https://nodejs.org/zh-cn/download/package-manager)

2. 使用npm安装依赖

在项目目录中打开终端，输入

```bash
npm install
```

如果安装速度比较慢，可以选择使用国内镜像

```bash
# 设置国内源
npm config set registry https://registry.npmmirror.com

# 查看国内源
npm get registry
```

3. 启动项目

在开发环境下

```bash
npm run dev
```
